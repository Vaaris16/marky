## ✨ Overview

**HyperFlux** is a next-generation frontend build tool designed for **speed**, **simplicity**, and **developer happiness**.  
It leverages native ES modules during development and an optimized production bundler under the hood.

---

## 🚀 Features

- ⚡ Instant dev server startup
- 🔥 Hot Module Replacement (HMR)
- 📦 Optimized production builds
- 🧠 Smart dependency pre-bundling
- 🧩 Plugin-based architecture
- 🛠 Framework-agnostic core
